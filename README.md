# Fake News Detection

A machine learning project that detects whether a news article is real or fake using Natural Language Processing (NLP) techniques.

## Features
- Preprocesses news text data
- Converts text into numerical vectors using TF-IDF
- Trains a Passive Aggressive Classifier
- Predicts whether a news article is REAL or FAKE
- Evaluates model accuracy

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK
- TF-IDF Vectorizer
- Passive Aggressive Classifier

## Dataset
- Fake.csv
- True.csv
